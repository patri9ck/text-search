    \section{Introduction}
    In this paper, we present the parallelization of text search in large texts for multiple queries in C++.
    The parallelization is done across the CPU, in threads and process and across the GPU.
    The paper also includes an approach which uses hierarchical approach, in which different processes are spawned
    which create threads which use the CPU\@.
    The goal is to find a solution per parallelization technology and also a combined solutions which works the best across
    most computers.


    \section{Testing Environment}
    For testing, we use 98 books in total to search in from Project Gutenberg and assemble it to one text which result in over 82 million characters.
    We query in total for the 100 most common english words.
    Compilation is done using G++ with O3 optimizations.
